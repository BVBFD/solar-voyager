# Texture Asset Pipeline

This directory is reserved for public-domain or permissively licensed astronomy assets.

Expected layout:

```text
textures/
  planets/<planet>/albedo.jpg
  planets/<planet>/normal.jpg
  planets/<planet>/clouds.png
  planets/<planet>/ring.png
  moons/moon/albedo.jpg
  moons/moon/normal.jpg
  backgrounds/milkyway.jpg
```

Do not add an asset without recording its source and license in the project README.
